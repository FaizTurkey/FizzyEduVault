# 🎓 FizzyEduVault

FizzyEduVault is a modern educational platform that helps students easily access and download Previous Year Question Papers (PYQs) for exam preparation.

## ✨ Features

- 📚 Download Previous Year Question Papers
- 🔍 Search and filter papers by subject or year
- 📱 Fully responsive design
- 🎨 Modern and clean UI
- ⚡ Fast loading performance
- 🌙 User-friendly experience

## 🛠️ Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS
- Node.js
- Express.js
